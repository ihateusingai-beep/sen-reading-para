# 老師使用手冊

> 對象：特殊教育老師（SEN）
> 所需時間：30 分鐘 setup + 每日 5 分鐘 dashboard 檢查
> 適用版本：Phase 4（v1.0+）

---

## 📦 1. 安裝

### 1.1 iPad 安裝
1. 開 Safari
2. 去 https://ihateusingai-beep.github.io/sen-yue-read-score/
3. 撳「分享」按鈕 → 「加入主畫面」
4. 個 App 會出現喺主畫面，似原生 App

### 1.2 設定離線（重要）
- iPad 第一次要連 WiFi，下載 Whisper 模型（~40MB，5 分鐘）
- 之後完全離線可用
- 喺 iPad 設定 → Safari → 進階 → 啟用「Experimental Features」可加強離線能力

### 1.3 多 iPad 部署
- 30+ iPad 都係用同一條 URL
- 教師機（你自己部 iPad）做管理
- 學生機（一人一機）做練習

---

## 👨‍🎓 2. 加學生（5 分鐘）

### 2.1 第一次
1. 開 App → 撳「🔧 老師模式」
2. 撳「👨‍🎓 學生」tab
3. 撳「＋ 新增學生」
4. 填：
   - **姓名**（例：小明）
   - **5 位 PIN**（例：12345，記住學生 PIN）
   - **頭像 emoji**（學生自己揀）
   - **顏色**（卡片邊色）
5. 撳「💾 儲存」

### 2.2 批次加學生
- 重複步驟 4，每次加一個
- 或者用「⬇️ 匯出 / ⬆️ 匯入」JSON 檔案（見 6.2 節）

### 2.3 揀登入模式
- **5 位 PIN** — 適合識數字嘅學生
- **名 + 頭像** — 適合不識字嘅學生
- **兩者皆可** — 學生自己揀

## ✏️ 3. 編輯詞彙

### 3.1 預設詞彙
App 內建 3 個分類：
- 自然季節（8 個詞）
- 戶外活動（5 個詞）
- 聲音世界（5 個詞）

### 3.2 開 / 關分類
1. 撳「🔧 老師模式」→ 「📋 設定」tab
2. 剔要學生見到嘅分類
3. 撳「✅ 儲存設定」

### 3.3 加新詞彙
1. 撳「✏️ 詞彙」tab
2. 喺要加詞嘅分類下面，撳「＋ 新增詞彙」
3. 填：
   - **字元**（例：蘋果）
   - **拼音**（例：ping gwo）
   - **詞義**（例：紅色水果）
   - **朗讀目標**（預設同字元）
   - **圖片檔名**（例：apple.png，要自己 push 到 `assets/images/`）
4. 撳「💾 儲存」

### 3.4 加新分類
1. 撳「✏️ 詞彙」tab
2. 撳「＋ 新增分類」
3. 輸入名稱（例：食物、顏色）
4. 完成！

### 3.5 圖片素材
- 圖片放喺 `assets/images/`
- 檔名要同 JSON 入面 `image` 欄一樣
- 建議正方形 200x200px，< 100KB

## ⚙️ 4. 揀 ASR 引擎

### 4.1 邊個引擎適合邊個情況

| 情況 | 建議引擎 |
|------|---------|
| 試用 / 唔想 setup | **Web Speech** |
| 真實使用 / 私隱 | **Whisper 本地** |
| 班房好嘈 / 唔需要標準音 | **自訂比對** |

### 4.2 設定 Whisper 本地
1. 撳「⚙️ 系統」tab
2. 揀「Whisper 本地」
3. 撳「⬇️ 預先下載 Whisper 模型」（要 5 分鐘，40MB）
4. 之後每次評估都用本地模型，零網絡、零私隱外洩

### 4.3 切換引擎
- 即時切換，唔使重新整理
- 設定記住喺每部 iPad localStorage
- 每部 iPad 獨立設定（唔使 sync）

## 📊 5. Dashboard

### 5.1 睇全班表現
1. 撳「📊 Dashboard」tab
2. 撳「全部 / 本月 / 本週」filter
3. 睇：
   - 學生人數
   - 練習次數
   - 可練習詞彙
   - **全班詞彙難點熱點**（邊 5 個詞全班都搞唔掂）

### 5.2 睇個別學生
1. 撳學生卡片「📋 詳細」
2. 睇：
   - 總練習次數
   - 逐個詞彙表現（✅ 掌握 / 👍 進步中 / ⚠️ 待加強）
   - 每次分數嘅迷你 bar chart
3. 撳「📄 匯出 PDF 報告」

### 5.3 匯出 PDF
- 學生個別報告：4 格統計 + 逐個詞彙表
- 全班報告：學生一覽表 + 全班難點 Top 10
- 撳「列印 / 存為 PDF」製 PDF 檔

## 💾 6. 備份 & 同步

### 6.1 詞彙備份
- 撳「✏️ 詞彙」tab → 「⬇️ 匯出 JSON」
- 得到 `words_2026-06-09.json`
- 撳「⬆️ 匯入 JSON」可以恢復

### 6.2 學生資料 + 進度備份
- 撳「👨‍🎓 學生」tab → 「⬇️ 匯出」
- 得到 `students_2026-06-09.json`
- **包含 PIN + 進度**
- 用喺跨 iPad 同步

### 6.3 跨 iPad 部署流程
```
老師 iPad: 加晒所有學生 + 設定 → 匯出 students_*.json
   ↓
派 JSON 檔案俾 30+ iPad
   ↓
每部學生 iPad: 撳「⬆️ 匯入」→ 即時有齊學生名單
   ↓
每部 iPad 獨立記錄練習進度
   ↓
收集 30+ iPad 嘅 students_*.json 匯入到老師 iPad → 統一睇全班
```

## 🔧 7. 常見問題

### Q1. 學生讀音正確但 ASR 認錯？
- A1. SEN 學童發音可能含糊。試下：
  - 鼓勵學生讀慢啲、大聲啲
  - 換用「自訂比對」引擎
  - 喺詞義入面用簡單詞

### Q2. Whisper 下載失敗？
- A2. 檢查網絡。iPad 第一次一定要連 WiFi。
- 撳「⚙️ 系統」→ 睇「Whisper 模型狀態」
- 失敗可重試或者切返 Web Speech

### Q3. 學生 iPad 唔識打字？
- A3. 個 App 主要用大掣、emoji 拼畫、唔需要打字。
- 「我的進度」頁只用睇，唔需要輸入。

### Q4. 私隱？
- A4. **完全本地、零雲端、零追蹤**。
- 學生語音永遠唔離開 iPad。
- 適合學校私隱政策要求。

### Q5. 點解唔用 Azure / 騰訊 API？
- A5. 收費 + 私隱 + 學生語音上載。
- 私隱係 SEN 學校嘅 hard requirement。
- Whisper 本地係最佳 trade-off：免費 + 私隱 + 廣東話友善。

## 🆘 8. 故障排除

| 問題 | 解法 |
|------|------|
| Safari 開唔到 | 試 Chrome / Edge |
| 冇聲音 | 檢查 iPad 靜音掣 + 麥克風權限 |
| 評估按鈕無反應 | 撳「⚙️ 系統」→ 切換引擎試下 |
| 學生進度冇咗 | 「👨‍🎓 學生」tab 撳匯出備份 |
| Whisper 慢 | iPad 唔夠力，試 base 換 tiny / 切 Web Speech |
